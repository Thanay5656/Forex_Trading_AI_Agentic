# Forex_Trading_AI_Agentic
# Forex Trading Analytics & Monte Carlo Risk Analysis Dashboard

##Project Overview

This project presents a comprehensive Forex Trading Analytics Dashboard developed using **Python** and **Microsoft Power BI**. It analyzes historical forex price data, trading performance, technical indicators, and Monte Carlo risk simulations to provide actionable insights for portfolio management and trading strategies.

The dashboard integrates data preprocessing, feature engineering, statistical analysis, and interactive visualizations for effective financial decision-making.

---

##Features

- Forex Market Analysis
- Trade Performance Analysis
- Monte Carlo Risk Simulation
- Interactive Power BI Dashboard
- Technical Indicator Analysis
- Risk Metrics Visualization
- Portfolio Performance Monitoring
